# Persona Chatbot

Minimal persona-based chat app with 3 personas.

- Anshuman Singh
- Abhumanyu Saxena
- Kshitij Mishra

## Built With

- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion (animations)
- React Markdown (message formatting)
- Gemini API (server route handler)

## Notes

- Requires `GEMINI_API_KEY` in `.env.local`.
- Persona prompts live in `prompts.md`.
